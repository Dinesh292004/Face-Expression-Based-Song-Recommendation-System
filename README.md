# 🎵 Face Expression Based Song Recommendation System

## 📌 Project Description

Face Expression Based Song Recommendation System is an AI-powered application that detects a user's facial expression in real-time using a webcam and recommends songs based on the detected emotion. The system enhances user experience by automatically selecting music that matches their mood.

---

## 🎯 Features

* 😊 Real-time face emotion detection
* 🎶 Automatic song recommendation based on mood
* 📷 Webcam integration
* 📊 Emotion tracking and history (optional)
* 🖥️ Simple and user-friendly interface

---

## 🛠️ Technologies Used

* Python
* OpenCV
* NumPy
* TensorFlow / Keras
* Tkinter (GUI)
* Pygame (for audio playback)

---

## ⚙️ Installation

1. Clone the repository:

```
git clone https://github.com/your-username/Face-Expression-Based-Song-Recommendation-System.git
```

2. Navigate to the project folder:

```
cd Face-Expression-Based-Song-Recommendation-System
```

3. Install required packages:

```
pip install -r requirements.txt
```

---

## ▶️ How to Run

Run the main file:

```
python main.py
```

---

## 🧠 How It Works

1. The system captures live video using a webcam
2. Detects the face using OpenCV
3. Predicts emotion using a trained deep learning model
4. Maps the emotion to a playlist
5. Plays songs that match the detected mood

---

## 😃 Supported Emotions

* Happy
* Sad
* Angry
* Neutral
* Surprise
